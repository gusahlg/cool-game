# A game I made for as a project in school

THE IDEA:
A strategy game about programming your units with an inbuilt scripting language with programmatic restrictions as resources.

Additional notes:
I did use claude code extensively for the making of the ahl-lang scripting language. This game was also submitted to the Brackeys game jam 2026.1.

To comipile and run the game you need to install Cargo and rust (a suggestion is to do this with rustup) then all you should have to do is to go to the game directory with the cd command and then write:
cargo run --release
This comamnd compiles the code as a release version (meaning that the compiler makes more optimizations than if it was compiled in debug mode), after compilation it is automatically run.
